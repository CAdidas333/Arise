Key Technical Notes

Zero dependencies is non-negotiable. No npm, no bundler, no frameworks. The entire game must remain a single index.html that works when double-clicked. This constraint drove every architecture decision.
Shadow AI engagement rule. Shadows only attack when the player is actively fighting (attacked within 3s) AND moving (moved within 10s). The "Arise: Legion" skill (key 5) overrides this for 12 seconds. Standing still for 3s+ also amplifies incoming damage up to 3x.
Dev mode is hidden. Backtick (`) = +5 levels, Shift+Backtick (~) = -5 levels. Not exposed in any UI. Useful for rapidly testing higher-level content without grinding.
Audio requires user gesture. Web Audio API won't play until the user interacts (browser policy). initAudio() is called on first keypress/tap and audioCtx.resume() is called when starting music.
